路由器udp端口转发1194到内网服务器192.168.1.224
路由器如果有固定IP地址，可以直接在脚本设置；
没有固定IP，需要开启花生壳之类的，域名在脚本设置IP时直接设置，也可以使用。

https://github.com/Nyr/openvpn-install

安装
wget https://git.io/vpn -O openvpn-install.sh && bash openvpn-install.sh
添加也使用此脚本
安装在/etc/openvpn/

服务配置文件
server.conf

local 192.168.1.224
port 1194
proto udp
dev tun
ca ca.crt
cert server.crt
key server.key
dh dh.pem
auth SHA512
tls-crypt tc.key
topology subnet
server 10.8.0.0 255.255.255.0
#push "redirect-gateway def1 bypass-dhcp"
#push "route 192.168.1.0 255.255.255.0 vpn_gateway"
push "route 192.168.1.0 255.255.255.0 vpn_gateway" //此设置让客户端只在访问VPN内时走VPN
ifconfig-pool-persist ipp.txt
push "dhcp-option DNS 114.114.114.114"
keepalive 10 120
cipher AES-256-CBC
user nobody
group nobody
persist-key
persist-tun
status openvpn-status.log
verb 3
crl-verify crl.pem
explicit-exit-notify

客户端模板clients-common.txt
client
dev tun
proto udp
remote 13998544800.wicp.vip 1194
resolv-retry infinite
nobind
persist-key
persist-tun
remote-cert-tls server
auth SHA512
cipher AES-256-CBC
ignore-unknown-option block-outside-dns // 以下两行删除 block-outside-dns OpenVPN 会添加 Windows 防火墙记录，拦掉除 tap 以外的所有网络接口上的 DNS 请求
block-outside-dns //删除
verb 3

关闭防火墙

添加iptables转发

iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT 
iptables -t nat -A POSTROUTING -d 192.168.1.0/24 -j SNAT --to 192.168.1.224

iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT 
iptables -t nat -A POSTROUTING -d 0.0.0.0/0 -j SNAT --to 78.141.218.222

查看转发
iptables -t nat -L

service iptables save

　　执行这个命令的时候有时候可能会报错：The service command supports only basic LSB actions (start, stop, restart, try-restart, reload, force-reload, status). For other actions, please try to use systemctl.

　　这是因为没有安装iptables服务，直接使用yum安装iptables服务即可.
yum install iptables-services
　　安装完成后，重新执行 service iptables save 命令即可保存成功。

　　配置iptables开机自启

　　保存后重启依然没有生效，后百度得知，需要设置iptables开机自启才可使配置生效。

　　执行如下命令（老版本命令为：service iptables on），设置iptables开机自启

systemctl enable iptables.service

注意：
　　需关闭firewalld防火墙

　　systemctl stop firewalld.service

　　systemctl disable firewalld.service

使用一下命令查看开启
systemctl list-unit-files|grep fire

添加开机启动（脚本写了）
systemctl enable –now openvpn-server@server.service

重启查看openserver状态
systemctl status openvpn-server@server.service
systemctl restart openvpn-server@server.service
